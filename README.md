# Read with Owly React Site

Готовый React-проект на Vite + Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

- `src/App.jsx` — основная страница сайта
- `src/main.jsx` — точка входа
- `src/index.css` — подключение Tailwind и базовые стили
- `tailwind.config.js` — конфиг Tailwind
- `vite.config.js` — конфиг Vite

## Что можно заменить

- тексты блоков
- ссылку кнопки покупки
- цвета и декоративные элементы
- изображения/обложку/маскота
